# Triniti Admin - Android App Releases

Репозиторий для релизов административной версии Android приложения Triniti.

## Установка

Скачайте последний APK файл из раздела [Releases](https://github.com/oracleh2/triniti-android-app-admin/releases).

## Версии

Все версии приложения доступны в разделе Releases с тегами формата `v1.0.0-admin`.

## Требования

- Android 8.0 (API 26) или выше
- Разрешение на установку из неизвестных источников
- Учётная запись администратора Triniti

## Changelog

История изменений доступна в описании каждого релиза.
